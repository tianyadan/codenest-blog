# 日程抽屉 · 忙碌时间轴组件

## 可复制提示词

```text
你是一名熟悉 Vue3 + TypeScript 的前端工程师。请在「日程创建/编辑抽屉」中实现 PC 端「忙碌时间轴」组件，并与左侧表单双向同步。

# 目标
用户除了用 DatePicker 精确输入起止时间外，还能在右侧时间轴上拖拽蓝色选中框的上下边界调整时段；同时看到参与人忙碌色块，便于避开冲突。

# 布局与端差异
- PC：抽屉双栏。左表单，右时间轴。
- 移动端：不展示时间轴；时间仅用表单 DatePicker。

# 时间轴交互
1. 纵轴为锚定日小时刻度，每小时固定高度（默认 48px）。
2. 蓝色选中框覆盖 startTime ~ endTime；上边改 start，下边改 end。
3. 指针 Y 转时间，按 15 分钟吸附；start < end，且至少间隔一个步长。
4. 结束时间可拖出当天底部，支持跨天。
5. 顶部提供前一天 / 今天 / 后一天；选中时段变化时尽量滚动到可见区域。

# 数据绑定
- 与表单 startTime、endTime 双向绑定；另有 anchorDate。
- 组件内统一 `YYYY-MM-DD HH:mm:ss`，提交前再转毫秒时间戳。

# 忙碌展示
- 参与人或范围变化时防抖请求忙碌接口。
- 半透明色块展示姓名 + 时段 + 主题；无权限显示「忙碌」。
- 支持图例筛选单人；忙碌块只读。
- 编辑时传 excludeEventId，避免把自己算进冲突。

# 验收
拖蓝框 ↔ DatePicker 双向同步；吸附正确；可跨天；移动端不渲染时间轴。
```

## 使用建议
