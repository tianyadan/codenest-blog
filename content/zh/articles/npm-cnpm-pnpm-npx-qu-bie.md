# npm,cnpm,pnpm,npx 区别

## 记录

npm 是 node.js 默认的包管理工具，也是最常用的。

cnpm 是淘宝将 npm 包从国外的服务器上拷贝到国内的服务器，使用方法与 npm 相同，但是加快了下载速度。

pnpm 意思是高性能的，最大的区别就是能够实现多个项目的包共享，不会每次都下载，节省电脑空间。

npx 可以允许在不安装全部工具包环境下临时运行命令，用完后会自动删除，适用于临时场景。
