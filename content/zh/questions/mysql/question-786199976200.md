## 答案

**回表** ： 指用二级索引查询时，因为<span style="color:#e74c3c">二级索引</span>的叶子节点只存<span style="color:#e74c3c">索引列的值</span>和<span style="color:#e74c3c">主键值</span>，拿不到其他字段的数据。如果查询需要其他字段，就得拿着主键再去<span style="color:#f39c12">聚簇索引</span>（主键索引）再去查一遍完整的数据行，这个过程就叫回表。

简单说： 二级索引不够用，得回到主键索引再查一次。

举个例子：


```SQL
select * from user where age = 20;
```
user 表有 id,name,age 三个字段，age 上有索引。

处理过程: 
1 ) 先走 age 索引，找到 age=20 的记录，拿到主键值。
2 ） 因为用了 select * ，需要name ，age ，id 三个字段，但 age 索引里没有 name 字段，得拿着主键去聚簇索引查完整数据。
3 ）最终拿到 id=2, name = 'xxx',age = 20 ;这条记录。
