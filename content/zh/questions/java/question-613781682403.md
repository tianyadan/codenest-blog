## 答案

1. public boolean equals（Object obj）

 用来比较两个对象是不是相等。默认是比较内存地址，也就是两个引用是不是同一个对象。
 
 实际开发中通常会重写这个方法，用来比较具体字段值，比如两个用户对象是否属于同一个人。
 
 2. public int hashCode（）
 
 返回当前对象的哈希值，主要给哈希类集合（HashMap、HashSet）用。如果重写了 equals 方法，这个方法也要一起重写，不然会出现看起来一样但是放不到一个桶里的问题。
 
 3. public String toString（）

返回对象的字符串形式。默认格式像是类名@十六进制哈希码，很难看。一般会重写，输出更具有意义的内容，比如现实某个关键字段。

4. public final Class<?> getClass（）

返回对象运行时的类信息。是 final 方法，不能修改。反射的时候经常用到，比如拿 class 名、构造方法等。

5. public void wait（）

当前线程挂起时，等着别的线程来 notify 它。也只能在同步代码块/同步方法中调用，不然会抛出异常。

6. public void notifyAll（）

和 notify 差不多，不过是唤醒所有在这个对象上等待的线程，谁能抢到锁谁先跑。

7. public void notify（）

唤醒一个正在等待当前对象的线程。必须在 synchronized 代码块中调用，配合 wait（）一起使用，做线程通信。
