## 答案

协程（Coroutine）是**一种轻量级的线程**，它允许在执行中暂停并在之后恢复执行，而无需阻塞线程。与线程相比，协程是用户态调度，**效率更高**，因为**它不涉及操作系统内核调度**。

Java 一开始没有原生支持协程，但在 Java19 之后通过 Project Loom 引入虚拟线程（Virtual Threads）最终在 Java21 中确认。它提供了类似于协程的功能。虚拟线程可以被认为是 Java 对协程的一种实现，虽然实现原理与传统协程略有不同，但它实现了高效并发。
