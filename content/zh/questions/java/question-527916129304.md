## 答案

**一句话总结**

**Java 泛型的作用**是通过在编译时检查类型安全，允许程序员编写更通用和灵活的代码，避免在运行时发生类型转换错误。

- 类型安全：泛型允许在编译时进行类型检查，确保在使用集合或其他泛型类时，不会出现类型不匹配的问题，减少了运行时的 ClassCastException 错误。

- 代码重用：泛型使代码可以适用于多种不同类型，减少代码重复，提升可读性和维护性。

- 消除显式类型转换：泛型允许在编译时指定类型参数，从而消除了运行时需要显示类型转换的麻烦。

但需要注意 Java 的泛型是伪泛型。
