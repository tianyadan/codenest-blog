## 答案

**内部类**是指**在一个类的内部定义的类**，Java 支持多种类型的内部类，**成员内部类**、**局部内部类**、**匿名内部类**、**静态内部类**。内部类可以访问外部类的**成员变量**和**方法**，甚至包括私有成员。

作用：
1. **封装性**： 将逻辑相关的类封装在一起，提高类的内聚性。
2. **访问外部类成员**： 内部类可以方便地访问外部类的成员变量和方法，尤其是在需要操作外部类对象的场景下非常有用。
3. **简化代码**：对于只在一个地方使用的小类，内部类嫩减少冗余代码，简化结构。
4. **事件处理**：匿名内部类广泛应用于实现回调函数或事件监听，简化了代码结构。尤其是对于实现接口或抽象类的场景。

---
类型：

1. **成员内部类**： 非静态类，作为外部类的一个成员，它可以直接访问外部类的所有成员，包括私有成员。
2. **静态内部类**： 定义为 `static` ，无法访问外部类的非静态成员，只能访问外部类的静态成员。
3. **局部内部类**： 定义在方法或代码块中的类，仅在该方法或代码块内可见，通用用于临时的对象创建。
4. **匿名内部类**： 没有类名的内部类，通常用于创建短期使用的类实例，尤其是在接口回调或事件处理时广泛使用。

---
举例：
1️⃣ 成员内部类（非静态）
成员内部类是外部类的一个成员，可以直接访问外部类的所有成员（包括私有成员）。
```
class Outer {
    private String msg = "Hello from Outer";

    class Inner {
        void show() {
            System.out.println(msg); // 直接访问外部类私有成员
        }
    }

    void test() {
        Inner inner = new Inner();
        inner.show();
    }
}

public class Main {
    public static void main(String[] args) {
        Outer outer = new Outer();
        outer.test(); // 输出：Hello from Outer
    }
}
```

---

2️⃣ 静态内部类
静态内部类不能访问外部类的非静态成员，只能访问静态成员。


```
class Outer {
    private static String staticMsg = "Hello from static Outer";
    private String instanceMsg = "Instance message";

    static class StaticInner {
        void show() {
            System.out.println(staticMsg); // 可以访问静态成员
            // System.out.println(instanceMsg); // ❌ 不允许访问非静态成员
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Outer.StaticInner inner = new Outer.StaticInner();
        inner.show(); // 输出：Hello from static Outer
    }
}
```
---

3️⃣ 局部内部类
局部内部类定义在方法内部，只能在方法内使用。

```
class Outer {
    void method() {
        class LocalInner {
            void greet() {
                System.out.println("Hello from LocalInner");
            }
        }

        LocalInner localInner = new LocalInner();
        localInner.greet();
    }
}

public class Main {
    public static void main(String[] args) {
        Outer outer = new Outer();
        outer.method(); // 输出：Hello from LocalInner
    }
}
```
---

4️⃣ 匿名内部类

匿名内部类没有名字，常用来快速创建接口或类的实例。


```
interface Greeting {
    void sayHello();
}

public class Main {
    public static void main(String[] args) {
        Greeting greeting = new Greeting() {
            @Override
            public void sayHello() {
                System.out.println("Hello from Anonymous Inner Class");
            }
        };

        greeting.sayHello(); // 输出：Hello from Anonymous Inner Class
    }
}
```
