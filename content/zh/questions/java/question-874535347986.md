## 答案

**永久代（Permanent Generation, PermGen）**

- 位置：永久代属于 方法区（Method Area）的一部分，在 Java 8 之前是 JVM 堆外的一块特殊内存区域。

- 存放内容：
- - 类的元数据（Class Metadata）：类的名字、修饰符、父类、接口、方法、字段等信息。 
- - 静态变量（static fields）。
- - 常量池（compile-time constants，比如 String 常量、final 常量等）。

- 特点：
- - 固定大小，必须通过 JVM 参数 -XX:PermSize 和 -XX:MaxPermSize 设置上限。
- - 容易出现 java.lang.OutOfMemoryError: PermGen space，尤其是在频繁加载、卸载类的场景（比如热部署的 Web 容器）。

💡 小技巧记忆：
- 永久代 = 永远占内存（固定，容易溢出）。
