## 答案

在 Java 中，可以中 **Runtime** 类或 **ProcessBuilder** 类来调用外部可执行程序或系统命令。这两种方法都能创建一个子进程来执行指定的命令或程序。


- 使用 Runtime.exec()

Runtime 类提供了exec()方法，它允许执行外部命令，使用相对 ProcessBuilder 比较简单。
例如：
```Java
try{
    // 执行命令
    Process process = Runtime.getRuntime().exec("notepad.exe");

    // 等待外部程序结束
    int exitCode = process.waitFor();

    // 检查执行结果
    if (exitCode == 0) {

        System.out.println("命令执行成功");
    } else {
        System.out.println("命令执行失败");
    }

    } catch (Exception e) {
        e.printStackTrace();
    }

```

- 使用 ProcessBuilder

ProcessBuilder 类提供了一个更灵活和更强大的方式来管理外部进程。它允许设置环境变量、工作目录、以及重定向输入和输出流。

 示例：
 
```Java

try{

    // 创建一个新的 ProcessBuilder 实例
    ProcessBuilder builder = new ProcessBuilder("cmd.exe","/c","dir");
    builder.redirectErrorStream(true); // 合并标准错误和标准输出
    builder.directory(new File("C:\\Windows")); // 设置工作目录

    // 启动进程
    Process process = builder.start();

    // 读取进程的输出
    InputStream is =   process.getInputStream();
    BufferedReader reader = new BufferedReader(new InputStreamReader(is));

    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }

    // 等待进程完成
    int exitCode = process.waitFor();

    if (exitCode == 0 ){
        System.out.println("命令执行成功");
    } else {
        System.out.println("命令执行失败");
    }

    } catch (Exception e) {
        e.printStackTrace();
    }


```
