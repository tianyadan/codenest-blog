## 答案

**元空间（Metaspace）**

**一句话概括：元空间是 Java 虚拟机中存储类元数据的内存区域，它取代了 Java 8 之前的永久代。**

- 引入时间：Java 8 开始，永久代被**彻底移除**，取**而代之的是 元空间**（Metaspace）。

- 位置：元空间不在虚拟机堆内，而是 使用**本地内存**（Native Memory）。

**存放内容：**

-  和永久代类似，主要存放**类**的元数据（Class Metadata）。
-  但 静态变量和常量池仍然在 Java 堆中。

**特点：**
- 默认可以动态增长，不再受 JVM 堆大小限制，避免了频繁出现 PermGen OOM。
- 可以通过参数 -XX:MetaspaceSize 和 -XX:MaxMetaspaceSize 来限制增长上限。
- 更容易与操作系统内存打交道。
