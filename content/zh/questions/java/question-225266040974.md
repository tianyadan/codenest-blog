## 答案

**序列化**就是把一个 Java 对象（在内存中的状态，比如属性值）转换成可以存储或传输的形式。通常是转换成 字节流（byte stream）。

存储场景：把对象写入文件、数据库、缓存
传输场景：对象通过网络在不同 JVM 之间传递

**反序列化**就是和序列化相反的过程：
把字节流重新“拆包”，还原成 Java 对象。
比如：从文件中读出一个对象；从网络接收到的字节流还原成对象

**java 序列化关键类和接口：**
**ObjectOutputStream**  用于序列化，**ObjectInputStream** 用于反序列化。类必须实现 **Serializable** 接口才能被序列化。

**transient** 关键字：在序列化过程中，有些字段不需要被序列化，例如敏感数据，可以使用 **transient** 关键字标记不需要序列化的字段。
