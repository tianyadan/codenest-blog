## 答案

**先说重点：**

多态是指**同一个接口**或**父类**引用变量可以指向不同的对象实例，并根据实际指向的对象类型执行相应的方法。

它允许同一方法在不同对象上表现出不同的行为，是面向对象编程（OOP）的核心特性之一。

**多态优点：**

通过多态，程序可以灵活地处理不同类型的对象，**降低代码耦合度**，**增强系统可扩展性**，新增子类或实现类时，无需修改原有代码，只需要通过接口或父类引用调用即可。


**通俗易懂讲：**

想象你去咖啡店点咖啡，咖啡店有很多种咖啡：美式、拿铁、卡布奇诺……

- 接口/父类：Coffee（咖啡）
- 子类/实现类：Americano、Latte、Cappuccino
- 方法：make()（制作咖啡）

**重点： 你作为顾客，只管“点咖啡”，不用关心咖啡师具体怎么做。这个“只管接口，不管具体实现”的思想，就是多态！**

Java 代码示例：


```
// 父类
abstract class Coffee {
    abstract void make(); // 制作咖啡
}

// 子类1
class Americano extends Coffee {
    @Override
    void make() {
        System.out.println("制作一杯美式咖啡");
    }
}

// 子类2
class Latte extends Coffee {
    @Override
    void make() {
        System.out.println("制作一杯拿铁咖啡");
    }
}

// 子类3
class Cappuccino extends Coffee {
    @Override
    void make() {
        System.out.println("制作一杯卡布奇诺");
    }
}

public class CoffeeShop {
    public static void main(String[] args) {
        Coffee coffee1 = new Americano(); // 父类引用指向子类对象
        Coffee coffee2 = new Latte();
        Coffee coffee3 = new Cappuccino();

        // 多态体现：同一个方法，不同对象表现不同
        coffee1.make(); // 输出：制作一杯美式咖啡
        coffee2.make(); // 输出：制作一杯拿铁咖啡
        coffee3.make(); // 输出：制作一杯卡布奇诺
    }
}
```
