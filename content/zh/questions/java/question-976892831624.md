## 答案

## 重点：

**在 java 中，参数传递只有按值传递，不论是基本类型还是引用类型。**

- **基本数据类型**（如:int,char,boolean,float 等）**传递**的是**值的副本**，即基本类型的数值本身。因此，对方法参数的任何修改都不会影响原始变量。
- **引用数据类型**（如对象引用）：传递的是引用的副本，即对象引用的内存地址。因此方法内可以通过引用修改对象属性，但不能改变引用本身，使其指向另一个对象。
