## 答案

循环依赖是指两个或多个模块、类、组件之间相互依赖，形成闭环。 换句话讲 A 依赖 B，B 又依赖 A，会导致 spring 懵逼，创建过程中发现 B 要依赖于 A，A 还没有创建好，无法确定加载或初始化的顺序。


```
@Service
public class A {
    @Autowired
    private B b;
}

```

```
@Service
public class B {
    @Autowired
    private A a;
}

```

 亦或者自己依赖自己，都会导致循环依赖，启动报错。
 
 ```
@Service
public class A {
    @Autowired
    private A a;
}

```
