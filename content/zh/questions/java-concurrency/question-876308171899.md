## 答案

**自定义拒绝策略**，可以实现 RejectedExecutionHandler 接口来自定义的拒绝策略。例如，记录日志或将任务重新排队。

```Java

public class CustomRejectedExecutionHandler implements RejectExecutionHandler {

@Override
public void rejectedExecution(Runnable r, ThreadPoolExecutor executor ) {

    System.out.println("task"+r.toString()+"rejected")
    // 可以在这里实现日志记录或其他逻辑

    }
}

```
