## 答案

**一句话**：CAS 是操作方式：**一种“比较并交换”的原子指令**。

CAS （Compare And Swap）是一个 CPU 原子指令，用来做：

“如果内存值 == 我的预期值 ，就把它改成新值，否则失败。”

**CAS 自旋锁示例：**


```Java
while (!compareAndSwap(markWord, old, new)) {
    // 再试？
}
```

CAS 本身不代表自旋，但你看代码就明白了：当 CAS 失败后，如果你继续 while 重试，那你就形成了 “自旋 + CAS”。

**🎯 关键关系：**

CAS 经常被放在自旋里面：**“CAS 自旋锁”**

这个其实就是：用 CAS 不断尝试抢锁 → 失败就继续循环 → 这就变成了“自旋 + CAS”组合
