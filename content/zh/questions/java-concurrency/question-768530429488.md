## 答案

CompleteableFuture 是 Java 8 引入的一个强大的异步编程工具。允许非阻塞地异步处理任务。并且通过链式调用组合多个异步操作。

核心特性：

1. 异步执行：使用runAsync() 或 supplyAsync()方法，可以非阻塞地执行任务。
2. 任务的组合：可以使用thenApply()、thenAccept()等方法在任务完成后进行后续操作，支持链式调用。
3. 异步处理：提供exceptyionally() , handle()等方法来异步处理任务中的异常。
4. 并行任务：支持多个异步任务的组合如,thenCombine(),allOf()等方法，可以在多个任务完成后进行操作。
5. 非阻塞获取结果，相比 Future，CompletableFuture 之后次通过回调函数获取结果，而不需要显示的阻塞等待。
